# Dlytful Data Map & Processing Record

**Last Updated:** December 2025

This document outlines the personal data collected, processing purposes, lawful basis, retention periods, and third-party processors for Dlytful, in compliance with GDPR and relevant privacy regulations.

## 1. Data Categories

We collect and process the minimum usage data necessary to provide the Dlytful service.

| Category | Description | Source |
| :--- | :--- | :--- |
| **Identity Data** | Name, email address, user ID. | Provided by user (Account creation). |
| **Billing Data** | Transaction history, plan status, Stripe Customer ID. **Note:** We do not store raw credit card numbers. | Generated by system / Provided by user via Stripe. |
| **User Content** | Brand tone inputs, generated copy assets, project configurations, and stored "vibes"/parameters. | Provided by user (Service usage). |
| **Technical Data** | IP Address, User Agent, Connection timestamps, API request logs. | Automated (System access). |

## 2. Purpose & Lawful Basis

| Purpose | Data Categories Used | Lawful Basis |
| :--- | :--- | :--- |
| **Service Provision** | Identity, User Content, Technical | **Contract**: Necessary to fulfill Terms of Service and deliver the SaaS product. |
| **Billing & Payments** | Identity, Billing | **Contract**: Necessary to process payments for subscriptions. |
| **Security & Fraud Prevention** | Identity, Technical | **Legitimate Interest**: Protecting the platform and users from abuse. |
| **Customer Support** | Identity, User Content | **Legitimate Interest / Contract**: Assisting users with issues. |

## 3. Data Retention

| Data Type | Retention Period | Deletion Policy |
| :--- | :--- | :--- |
| **Active Account Data** | Indefinite while account is active. | Deleted upon user request or account closure. |
| **Deleted Account Data** | 30 days post-deletion grace period. | Hard deleted from production database after grace period. |
| **System Logs** | 90 days. | Automatically rotated/deleted. |
| **Backups** | 7-30 days (Rolling point-in-time recovery). | Overwritten automatically. |

## 4. Third-Party Processors (Sub-processors)

We use the following third-party service providers to process data on our behalf. All providers are vetted for security and compliance.

| Processor | Purpose | Location |
| :--- | :--- | :--- |
| **Supabase** | Database, Authentication, Real-time subscriptions. | AWS (Region dependent, typically US/EU). |
| **Vercel** | Web hosting, Edge Functions, Content Delivery Network. | Global (Edge Network). |
| **Stripe** | Payment processing and billing management. | Global / US. |
| **Google Gemini / OpenAI** | LLM inference for copy generation services. | US / Global. |

***

**Note:** Dlytful does not currently use non-essential cookies, third-party analytics (e.g., Google Analytics), or marketing tracking pixels.
